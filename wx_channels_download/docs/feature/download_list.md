# 下载列表功能

## 功能说明

在视频号页面添加了一个悬浮的下载列表，可以查看和管理所有下载记录。

## 使用方法

### 打开下载列表

1. **右键点击悬浮下载按钮**：在页面右下角的绿色"下载当前视频"按钮上右键点击
2. **长按悬浮下载按钮**：长按 0.5 秒
3. **点击列表图标**：在悬浮下载按钮左侧有一个 📋 图标，点击即可打开

### 列表功能

- **查看下载记录**：显示所有已下载、正在下载和失败的视频
- **状态标识**：
  - 🟢 下载中：视频正在下载
  - 🔵 已完成：视频下载完成
  - 🔴 失败：下载失败
- **操作按钮**：
  - **重新下载**：对已完成的视频可以重新下载
  - **复制命令**：复制下载命令到剪贴板，可在终端使用
  - **重试**：对失败的下载可以重试
  - **删除**：从列表中移除记录

### 列表特性

- **自动记录**：每次点击下载按钮时，会自动添加到列表
- **状态更新**：下载状态会自动更新
- **折叠/展开**：点击标题栏可以折叠或展开列表
- **清空列表**：点击"清空"按钮可以清空所有记录
- **最大显示**：最多显示 10 条记录，超出会自动删除最旧的记录

## 界面说明

列表显示在页面右下角，悬浮下载按钮的上方：
- 标题栏显示"下载列表"和完成数量统计
- 每个列表项显示：
  - 视频文件名/标题
  - 下载状态（下载中/已完成/失败）
  - 视频规格（如 xWT111、原始等）
  - 下载时间
  - 操作按钮

## 注意事项

1. 列表数据仅保存在当前页面，刷新页面后会清空
2. 下载状态更新可能有延迟，请耐心等待
3. 如果下载失败，可以点击"重试"按钮重新下载
